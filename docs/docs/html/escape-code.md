# HTML 转义字符

---

## 常用 HTML 实体

| 符号 | 实体名称 | 实体编号 | 说明 |
| --- | --- | --- | --- |
| `&` | &amp; | &#38; | 连接符号、and 操作符 |
| `$` | &dollar; | &#36; | 美元符号 |
| `€` | &euro; | &#8364; | 欧元符号 |
| `£` | &pound; | &#163; | 英镑符号 |
| `←` | &larr; | &#8592; | 左箭头 |
| `↑` | &uarr; | &#8593; | 上箭头 |
| `→` | &rarr; | &#8594; | 右箭头 |
| `↓` | &darr; | &#8595; | 下箭头 |
| `↔` | &harr; | &#8596; | 左右箭头 |
| `↵` | &crarr; | &#8629; | 回车符号 |
| `‹` | &lsaquo; | &#8249; | 左单角引号 |
| `›` | &rsaquo; | &#8250; | 右单角引号 |
| `«` | &laquo; | &#171; | 左双角引号 |
| `»` | &raquo; | &#187; | 右双角引号 |
| `•` | &bull; | &#8226; | 着重号 |
| `…` | &hellip; | &#8230; | 省略号 |
| `×` | &times; | &#215; | 乘号 |
| `–` | &ndash; | &#8211; | 短破折号 |
| `—` | &mdash; | &#8212; | 长破折号 |
|  | &ensp; | &#8194; | 半角空格 |
|  | &emsp; | &#8195; | 全角空格 |
|  | &thinsp; | &#8201; | 窄空格 |
| `™` | &trade; | &#8482; | 商标 |
| `®` | &reg; | &#174; | 注册商标 |
| `©` | &copy; | &#169; | 版权符 |
